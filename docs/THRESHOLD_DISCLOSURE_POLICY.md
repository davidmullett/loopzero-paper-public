# Threshold Disclosure Policy

This repository publishes coarsened, publication-grade threshold regimes for the witness classes used in the manuscript.

The purpose of this file is to support the manuscript's scientific claim that the detector occupies a stable threshold regime across preregistered domains, while avoiding disclosure of exact production tuning surfaces, calibration ladders, or deployment-specific threshold maps.

Accordingly:

- The public threshold artifact reports broad, rounded ranges appropriate for scientific communication.
- It does not expose exact per-domain production values, tolerance maps, calibration search procedures, or operational tie-break logic.
- The public ranges are sufficient to interpret the manuscript's threshold-stability claim and to reproduce the publication-grade analyses documented in this repository.
- The public ranges are not intended to document or substitute for non-public production deployment thresholds.

This repository is publication-complete and production-incomplete.
