# No-Progress 3-Cycle: Public Theorem Notes

The formal core of the manuscript is a minimal obstruction over an abstract preorder of system states.

In scientific terms, the obstruction can be stated as follows:

- let x < y denote a strict progress relation,
- let y <= z and z <= x denote two monotone return constraints,

then the resulting composition forces contradiction.

Intuitively, the system exhibits one strict leg of progress together with a recursive return path that tethers the trajectory back toward its starting state. Once strict progress and monotone return are composed in this way, the cycle cannot remain coherent.

This public note is intended to support the theorem-grounded scientific framing of the paper. It does not attempt to disclose additional proof-engineering details beyond what is necessary for scientific interpretation.
